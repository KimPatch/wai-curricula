## Introduction
{:.no-display}

This module introduces accessibility considerations related to images in web content.

## Learning Outcomes for Module

* define informative Images as those that graphically represent content and information.
* Define decorative images as those whose purpose is to add visual decoration to the page.
* Define functional images as those who represent a specific functional purpose.
* Define images of text as those which represent only pieces of written text but coded as an image.
* Define complex images as those which represent data or complex information.
* Provide adequate alternative texts for the above type(s) of image(s).
